# Annabelle Final Product Kit

This package turns Annabelle's current AI music-transcription coursework into three more polished final deliverables:

1. `paper/ANNABELLE_PUBLISHABLE_PAPER_PLAN.md` — a paper plan that reframes her work as a publishable student research brief.
2. `docs/` — a GitHub Pages-ready static website. Copy this folder into the root of `annabelle-z-li/AI-research-level-2`, then enable Pages from `main` → `/docs`.
3. `huggingface-space/` — a more functional Gradio app that combines transcription, scoring, downloadable artifacts, and a musician-centered report.

## Suggested final-session workflow

1. Deploy an instructor-owned model version first so Annabelle can see the destination.
2. Help Annabelle add the `docs/` folder to her GitHub repository and revise the language in her own voice.
3. Help her duplicate or create a new Hugging Face Space and upload the files from `huggingface-space/`.
4. Ask her to revise `PAPER.md` using the paper plan, especially the methods/results/discussion sections.

## Deployment note

This bundle is safe to deploy under the instructor's account as a clearly labeled model/demo. It should not be represented as Annabelle's finished student-authored submission until she revises it, runs her own tests, and links her own final artifacts.
